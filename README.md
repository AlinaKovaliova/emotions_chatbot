# emotions_chatbot

# Структура проекта

    ├── 📂 backend
    │   ├── chatbot_detailed.ipynb   # Jupyter Notebook с подробным описанием этапов разработки
    │   └── chatbot_final.ipynb      # Jupyter Notebook с финальным кодом чат‑бота в одной ячейке
    ├── 📂 flowchart
    │   ├── chatbot_flowchart.png    # Блок‑схема логики работы чат‑бота (PNG)
    │   └── chatbot_flowchart.drawio # Оригинальный файл блок‑схемы для редактирования
    └── README.txt                   # Текущий файл с описанием содержимого, инструкцией по запуску и критериями выбора моделей

## Инструкция по запуску

1. Убедиться, что установлены библиотеки `transformers`, `torch`, `deep-translator`, `pandas`.  
   Для установки выполнить команду:  
   `!pip install transformers torch deep-translator pandas`

2. Открыть `chatbot_final.ipynb`.

3. Запустить единственную ячейку кода.

4. Ввести тестовое сообщение пользователя в поле `input` и нажать Enter, проанализировать корректность определения эмоций и выдаваемых рекомендаций.

5. Для выхода ввести `"выход"` или `"exit"`.

6. Для подробного разбора алгоритма работы открыть `chatbot_detailed.ipynb`.

7. Ознакомиться с логикой работы чат‑бота с помощью `chatbot_flowchart.png` / `chatbot_flowchart.drawio`.

## Критерии выбора моделей

Для задачи классификации эмоций была выбрана модель **`bhadresh-savani/distilbert-base-uncased-emotion`** из Hugging Face, так как она:

- Показывает хорошие результаты на задаче классификации эмоций;
- Балансирует между скоростью работы и качеством результатов;
- Лёгка в интеграции в Jupyter Notebook без необходимости дополнительного дообучения.

Для перевода текста на английский язык использовалась библиотека `deep-translator` с сервисом Google Translate. Это необходимо, так как модель обучена на англоязычных данных. Данная модель позволяет корректно переводить русскоязычные тексты в английские для улучшения качества классификации эмоций.
